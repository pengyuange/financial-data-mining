	TWNN.jar
Company stock data retreival

Requirements:
* Java Runtime

To run:
Open command prompt and locate the MQP 
directory. Then, in the command prompt, type:

java -jar twnn.jar

This will launch the application. If it is your
first time launching the application, you will
need to first import company data. master.csv includes
over 5000 stock symbols that you can import into the
database.
